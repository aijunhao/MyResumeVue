## 项目介绍

`MyResumeVue` 是 “我的简历” 网站设计的前端部分，项目整体采用 `Vue-Cli 3` 工具进行构建，使用 `Element-UI` 快速成型工具进行主要开发。

## 项目配置

```
npm install
```

### 项目编译运行
```
npm run serve
```

### 打包项目
```
npm run build
```


### 自定义配置

见 [Vue-Cli](https://cli.vuejs.org/config/).
